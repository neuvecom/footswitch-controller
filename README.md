# footswitch-controller

PC Sensor FS23（3ボタン USB フットスイッチ）用の macOS メニューバー常駐コントローラーです。

フットスイッチが送るキー入力を横取りして、アクティブなアプリやユーザー定義のモードごとに、任意のアクション（キー送出・テキスト入力・URL 起動・スクリプト実行など）に置き換えます。

## 主な機能

- メニューバー常駐（Dock アイコンなし）
- 3 ボタンそれぞれに任意のアクションを割り当て
  - キーストローク送出（修飾キー対応）
  - テキストを打つ（任意の文字列をそのまま入力、`\n` などの記号も可）
  - URL を開く / Shell スクリプト / AppleScript の実行
  - モード切替
- アプリごとの割り当て + 1 アプリ内の複数モード切替
- 設定は自動保存
- アップデート確認（新バージョンがあれば通知）

## 動作環境

- macOS 13.0 (Ventura) 以降
- Apple Silicon（M1 以降）
- **アクセシビリティ権限**が必要（キー入力の検知と送出に使用）

## 対応機種

- PC Sensor FS23（3 ボタン USB フットスイッチ）
  - 購入先: [Amazon.co.jp](https://www.amazon.co.jp/dp/B0FRMVL5SP)

各ボタンは固定のキー（F13 / Option+F13 / Ctrl+F13）を送出し、本アプリがそれを検知して任意のアクションへ置き換えます。

## 初期設定（フットスイッチの焼き込み）

本アプリは、フットスイッチの各ボタンが以下のキーを送る前提で動作します。出荷時の設定は異なる場合があるため、メーカー製ソフト **Elfkey** で先に各ペダルへ焼き込んでください。

| ボタン | 焼き込むキー |
| --- | --- |
| 1 | F13 |
| 2 | Option (⌥) + F13 |
| 3 | Control (⌃) + F13 |

手順:

1. [pcsensor.com/download](https://pcsensor.com/download/) から Elfkey をダウンロードします（macOS は Apple Silicon / ARM 版を選択）。
2. フットスイッチを USB 接続し、Elfkey を起動します（グラフィカルな設定画面です）。
3. 各ペダルに上表のキーを割り当て、デバイスへ書き込みます。
4. **コツ: F13 は Mac の物理キーボードでは押せません。** Elfkey 画面内の**ソフトウェアキーボード**から F13 を入力してください。Option / Control の修飾は、修飾キーと組み合わせて指定します。

焼き込みが終わったら、下記のアプリ本体をインストールします。

## インストール

1. [Releases](https://github.com/neuvecom/footswitch-controller/releases/latest) から `FootswitchController.dmg` をダウンロードします。
2. DMG を開き、`FootswitchController.app` を `Applications` フォルダにドラッグします。
3. アプリを起動します（Developer ID 署名 + Apple 公証済みなので、そのまま開けます）。
4. 初回起動時、アクセシビリティ未許可ならオンボーディング画面が表示されます。案内に従って **システム設定 → プライバシーとセキュリティ → アクセシビリティ** で本アプリを許可してください。
5. 許可は自動で検知され、付与した瞬間に監視が始まります（アプリの再起動は不要）。

メニューバーのアイコンから設定画面を開き、各ボタンにアクションを割り当てて使います。
